# Heartly AI Landing Page

## SEO & Psychology-Optimized Landing Page for Skin Analysis App

This landing page is built using conversion psychology principles and SEO best practices.

---

## 🧠 Psychological Triggers Used

### 1. **Urgency & Scarcity**
- "50% descuento para primeros 1,000 usuarios"
- "Quedan 847 lugares"
- Creates FOMO (Fear Of Missing Out)

### 2. **Social Proof**
- "2,847 mujeres ya analizaron su piel esta semana"
- Avatar stack showing real users
- Testimonials with photos
- 5-star ratings

### 3. **Loss Aversion**
- "Mira cómo lucirás en 5 años SI NO te cuidás"
- Fear of aging poorly drives action

### 4. **Hope & Gain Framing**
- "Si te cuidás: 22 años" (younger than actual age)
- Positive future visualization

### 5. **Comparison & Competition**
- "¿Quién tiene mejor piel? Desafía amigas"
- Social comparison drives engagement

### 6. **Curiosity Gap**
- "Tu piel tiene una edad. Probablemente no es la que crees."
- User NEEDS to know their "real" age

### 7. **Authority**
- "Análisis con IA"
- "468 puntos de tu rostro"
- Scientific/technical language

### 8. **Simplicity**
- "3 pasos, 30 segundos"
- Low barrier to entry

## 📊 SEO Optimization

### Technical SEO
- ✅ Semantic HTML5 structure
- ✅ Meta title (60 chars)
- ✅ Meta description (155 chars)
- ✅ Canonical URL
- ✅ Open Graph tags (Facebook)
- ✅ Twitter Card tags
- ✅ Schema.org structured data
- ✅ Mobile-responsive design
- ✅ Fast loading (no heavy JS)

### Keyword Strategy
**Primary keywords:**
- "análisis de piel"
- "edad de la piel"
- "análisis facial IA"
- "edad aparente"
- "skincare personalizado"

**Secondary keywords:**
- "cuidado de la piel"
- "piel saludable"
- "análisis de rostro"
- "rutina skincare"

### Content SEO
- H1: Main value proposition with primary keyword
- H2: Section headers with secondary keywords
- Alt text on all images
- Internal linking structure
- Clear content hierarchy

## 🚀 Deployment

### Option A: Netlify (Recommended)
```bash
# Install Netlify CLI
npm install -g netlify-cli

# Deploy
cd landing
netlify deploy --prod
```

### Option B: GitHub Pages
```bash
# Push to GitHub
git add landing/
git commit -m "Add SEO-optimized landing page"
git push

# Enable GitHub Pages in repo settings
```

### Option C: Vercel
```bash
# Install Vercel CLI
npm install -g vercel

# Deploy
cd landing
vercel --prod
```
